# 🛍️ Community-Store

A modern, responsive **community-driven marketplace web application** where users can buy, sell, and discover products within their local community.

---

## 📌 Project Overview

**Community-Store** is a front-end web project designed to simulate a real-world e-commerce platform focused on **local trading and community engagement**.

The platform allows users to:
- Browse products  
- Sell items  
- Add items to cart  
- View profiles  
- Interact with a marketplace-style UI  

This project demonstrates **UI/UX design, responsive web development, and modern CSS structuring**.

---

## 🎯 Objectives

- Create a clean and user-friendly marketplace interface  
- Apply modern CSS (Flexbox, Grid, Variables)  
- Build a responsive website for different screen sizes  
- Simulate real-world e-commerce features  
- Strengthen front-end development skills  

---

## 🚀 Features

### 🏠 Home Page
- Hero section with video background  
- Promotional banners  
- Featured products grid  
- Scrolling announcements and ticker  

### 🛒 Product System
- Product cards with:
  - Price  
  - Ratings  
  - Seller info  
- Add to cart functionality (UI-based)  
- Product modal (detailed view)  

### 🧑‍🤝‍🧑 Community Features
- Vendor showcase section  
- Bulletin board (announcements)  
- Community-driven marketplace feel  

### 🛍️ Cart System
- Slide-in cart drawer  
- Quantity controls  
- Total calculation UI  

### 📦 Sell Product Page
- Form to upload product details  
- Image upload UI  
- Price input styling  

### 👤 Profile Page
- User avatar  
- Tabs for navigation  
- Seller role display  

### 🔐 Authentication UI
- Login / Register modal  
- Tab switching between forms
